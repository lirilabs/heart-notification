🖥 How server-level scheduling works

        Times you set here are saved to Firestore (config/schedules).
        Vercel runs /api/cron/push every 30 minutes automatically.
        Each run checks if the current UTC time matches any enabled slot (±28 min window).
        If it matches, the batch fires. The HTML page does not need to stay open.
        
Also: duplicate devices (same FCM token across different user accounts) receive only one notification.
